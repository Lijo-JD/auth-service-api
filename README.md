# Auth Service API

A secure authentication and authorization backend built with **Node.js**, **Express**, and **TypeScript** — designed to provide a robust authentication layer for web and mobile applications.

## 🚀 Features

- **User Registration & Login**
- **JWT-based Authentication**
- **Secure Password Hashing**
- **Middleware for Protecting Routes**
- Modular and TypeScript-friendly architecture
- Environment-based configuration

## 🧱 Tech Stack

| Technology | Purpose |
|------------|---------|
| **Node.js** | Server runtime |
| **Express** | Web framework |
| **TypeScript** | Type safety |
| **JWT** | Token-based authentication |
| **bcrypt** | Password hashing |
| **dotenv** | Environment variable management |
